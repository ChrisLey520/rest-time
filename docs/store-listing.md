# 栖时应用市场素材

AGC 提交时直接取用。配合 docs/release-checklist.md 第 3 节。

## 基础信息

- 应用名称：栖时
- 分类建议：效率 / 时间管理
- 一句话简介（30 字内）：30 秒进入下一段专注，把时间回收成进度和复盘。

## 完整描述

栖时是一款本地优先的专注计时应用，为需要自主管理时间的人设计——自由职业者、学生、研发、写作者、备考者。

它回答三个问题：现在该做哪一段？这段时间怎么守住？做完之后时间去了哪里？

【30 秒开始下一段】
打开就有下一段建议，附推荐理由和本段收益。没有任务也能直接开始自由专注，或写下一件事立即开始。

【状态不好也能启动】
最近中断偏多时，不硬扛完整专注——5 分钟保底启动、10/15 分钟稳住启动。缺席几天回来，迎接你的是一句「回来了就好」和一个 5 分钟的台阶，而不是清零的数字。

【专注中有守护】
专注一开始，离开 App、切后台、锁屏都会让本轮失败——在场才算数。想中途退出时，会先问原因、给 20 秒冷静期，卡住了就把任务拆成更小的 3 分钟动作。真的放弃也会留下交接，下次不从空白开始。

【失败有救援】
放弃后回来，1/3/5 分钟恢复阶梯帮你重新接上节奏。专注节奏采用弹性规则：每 7 天允许 1 个休息日，节奏不断。

【时间会被回收】
完成的每一段自动变成任务进度、今日目标、项目归属和月度复盘。完整守住的专注获得时光币，可兑换节奏保护券和专注氛围主题。

【复盘与交付】
月度报告展示投入趋势、最佳连续、高效时段和项目分布，可导出 CSV 和长图。自由职业者可按项目生成客户账单：费率、取整、不计费标记、发票备注。

【数据只属于你】
所有数据仅保存在设备本地，无账号、无上传、无第三方 SDK。卸载前可用导出功能备份。

- 时光币文案统一用「成就/兑换」，避免「货币/充值」表述（无支付功能，纯本地成就）。
- 隐私标签选「不收集数据」；权限说明：代理提醒 = 专注到点提醒。
- 描述中不出现「最」「第一」等绝对化用语（广告法）。
- 截图不含第三方商标、真实姓名和敏感项目名（用「客户 A / 写作 / 备考」等示例数据）。
